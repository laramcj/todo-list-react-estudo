# Todo List React - Estudo

Este é um projeto de estudo para criar uma aplicação de lista de tarefas (To Do List) utilizando React. O objetivo é praticar conceitos fundamentais do React, como componentes, estado e manipulação de eventos.

## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para construção de interfaces de usuário.
- **Vite**: Ferramenta de build rápida para desenvolvimento moderno.
- **ESLint**: Ferramenta de linting para manter a qualidade do código.

## Instalação

1. Clone o repositório:
   ```
   git clone <url-do-repositorio>
   cd todo-list-react-estudo/todo-list-react
   ```

2. Instale as dependências:
   ```
   npm install
   ```

## Uso

Para executar a aplicação em modo de desenvolvimento:

```
npm run dev
```

A aplicação estará disponível em `http://localhost:5173` (porta padrão do Vite).

### Outros Scripts Disponíveis

- `npm run build`: Compila a aplicação para produção.
- `npm run lint`: Executa o linter para verificar erros no código.
- `npm run preview`: Visualiza a versão de produção localmente.

## Estrutura do Projeto

- `src/App.jsx`: Componente principal da aplicação.
- `src/main.jsx`: Ponto de entrada da aplicação.
- `src/App.css` e `src/index.css`: Estilos da aplicação.
- `public/`: Arquivos estáticos.

## Funcionalidades

- Adicionar novas tarefas.
- Marcar tarefas como concluídas.
- Remover tarefas.

Este projeto é parte de um estudo pessoal para aprender React.
